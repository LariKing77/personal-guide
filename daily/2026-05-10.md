# День 2026-05-10 (Воскресенье)

## Тактика дня

- **Окно:** 12:00, 25+5 мин (одно конкретное время из weekly-слотов).
- **Действие:** запустить трекер времени → выбрать любую категорию из 3 (Planora / job-search / Развитие) → 25 мин одной задачи → 5 мин перерыв.
- **Минимум:** один слот закрыт. Не «хорошо», а «закрыт».

## Что подойдёт под слот сегодня (на выбор)

- Дочитать/доработать `DS-strategy/drafts/one-pager-tex-planora.md` (WP-007, in_progress) — 25 мин на одну секцию.
- Запустить bootstrap одного из PACK-* репо (WP-006).
- Прочитать 5-7 страниц чего-то системного (SoTA, Pack-знание).

Выбор — за тобой. Главное: **не «что важнее»**, а **«один слот в 12:00»**.

## Критерий закрытия дня

Слот отмечен в трекере. Запись `daily/2026-05-10.md` обновлена в этом репо строкой:
- ✅ слот 12:00–12:25 — `<категория>` — `<что делал>`
- ❌ слот пропущен — `<честная причина: не запустил, отвлёкся, не до этого>`

**Любая** запись лучше пустой. Пропуск с честной причиной — материал для рефлексии в `weekly/2026-W19.md`.

## Вопрос для чата с Claude Code в конце дня

«Слот закрыт / не закрыт — какое объяснение точнее описывает сегодня: ресурс, приоритет, ритм?»

- **ресурс** = устал, выгорел, не было сил
- **приоритет** = были задачи срочнее
- **ритм** = просто не вспомнил / не подошло время

Ответ копится в `history/` и используется для корректировки следующего дня.
